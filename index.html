Why ConDABench?

A robust evaluation framework designed to assess language model performance across diverse data analysis scenarios.

Comprehensive Evaluation

Evaluate language models across multiple data analysis tasks including statistical analysis, visualization, and pattern recognition.

Real-World Datasets

Built on authentic CSV datasets from finance, healthcare, demographics, and other domains for practical assessment.

Interactive Framework

Explore benchmark results through an intuitive interface with detailed analysis views and performance metrics.

Standardized Metrics

Compare models using consistent evaluation criteria including PASQ, ABQ, and UASQ metrics for reliable benchmarking.

Open Source

Fully open-source framework enabling researchers to reproduce results and extend the benchmark with new datasets.

Easy Integration

Simple API and pipeline setup for evaluating both open-source and proprietary language models on data analysis tasks.
